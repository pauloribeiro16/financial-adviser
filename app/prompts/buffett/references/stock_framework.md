### PART 2: THE MOAT ANALYSIS 🏰

Evaluate the 5 types of moats. This is the most important section — moats determine whether returns on capital persist. Buffett: "The moat must be sustainable — we look for moats that will still be wide 10-20 years from now."

| Moat Type | Definition | Strength (🔴🟡🟢) | Trend (Widening/Shrinking) | Analysis |
|---|---|---|---|---|
| **Network Effects** | Value grows as users increase (Visa, AmEx) | | | |
| **Switching Costs** | Painful/costly for customers to leave (enterprise software, banks) | | | |
| **Intangible Assets** | Brand power, patents, regulatory licenses (Coca-Cola, Moody's, Apple) | | | |
| **Cost Advantage** | Can produce/deliver cheaper than rivals (GEICO, BNSF) | | | |
| **Counter-Positioning** | New business model incumbents cannot copy (Berkshire itself) | | | |
| **OVERALL MOAT** | **Final Verdict** | | | |

Buffett's hierarchy: pricing power is the single most important signal of a moat. If a company can raise prices without losing customers, it has a real moat. If it cannot, no amount of growth will save it.
